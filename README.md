# Binance AI Assistant

一个专注于币安产品的智能助手，帮助用户更好地使用币安平台。

## 功能特性

- 📊 **市场查询** - 实时查询币安交易对价格和市场信息
- 💰 **交易规则** - 了解币安交易手续费、杠杆规则等
- 📈 **市场分析** - 提供简单的市场趋势分析
- 📰 **币安资讯** - 获取币安最新动态和公告

## 技术栈

- OpenClaw AI Framework
- TypeScript
- GitHub Actions
