# Binance AI Assistant

一个功能强大的币安产品智能助手，基于OpenClaw AI Framework构建，提供全方位的币安服务支持。

## 🎯 核心功能

### 1. 📊 市场数据查询
- **实时价格查询**：查询任意交易对的最新价格
- **24小时行情**：查看价格涨跌幅、最高价、最低价、成交量等
- **K线数据**：获取不同时间周期的K线数据（1m, 5m, 15m, 1h, 4h, 1d）
- **深度图数据**：查看买卖盘深度
- **最新成交**：获取最新成交记录
- **资金费率**：查询永续合约资金费率

### 2. 💱 交易规则查询
- **手续费率**：查询不同交易对的手续费率
- **杠杆规则**：了解杠杆倍数限制
- **交易对信息**：查询交易对状态、最小交易量等
- **合约规则**：了解合约交易规则

### 3. 📈 市场分析
- **价格趋势分析**：基于K线数据提供趋势判断
- **技术指标**：计算并显示RSI、MACD、布林带等技术指标
- **成交量分析**：分析成交量变化趋势
- **市场情绪**：基于资金费率判断市场情绪

### 4. 📰 币安资讯
- **最新公告**：获取币安官方公告
- **活动信息**：查询币安举办的各类活动
- **产品更新**：了解币安新功能上线

### 5. 🎮 交易辅助
- **价格提醒**：设置价格提醒（需要API Key）
- **持仓查询**：查询当前持仓情况
- **订单查询**：查询订单状态
- **交易建议**：基于市场数据提供交易建议（仅供参考）

## 🚀 技术架构

### 技术栈
- **OpenClaw AI Framework**：AI对话框架
- **TypeScript**：类型安全的开发语言
- **Express.js**：Web服务器框架
- **axios**：HTTP客户端
- **node-cache**：缓存层，提高响应速度

### API集成
- **币安公共API**：获取市场数据
- **币安交易API**：执行交易操作（需要API Key）
- **币安WebSocket**：实时数据推送

### 数据流
```
用户输入 → AI理解意图 → 调用对应API → 数据处理 → AI生成回复 → 返回用户
```

## 🔧 配置说明

### 环境变量
```bash
# 币安API配置（可选，用于交易功能）
BINANCE_API_KEY=your_api_key
BINANCE_API_SECRET=your_api_secret

# 服务器配置
PORT=3000
NODE_ENV=development
```

### API Key权限
- **读取权限**：查询市场数据、持仓、订单等
- **交易权限**：挂单、取消订单、查询订单等
- **提现权限**：充值、提现（谨慎使用）

## 📊 数据缓存

为了提高响应速度，系统使用以下缓存策略：

- **市场数据缓存**：5分钟
- **K线数据缓存**：1分钟
- **资讯数据缓存**：30分钟
- **交易规则缓存**：1小时

## 🔒 安全性

- **API Key加密存储**：使用环境变量或加密配置文件
- **请求签名**：所有交易请求都经过签名验证
- **IP白名单**：支持设置API Key白名单
- **请求频率限制**：防止API滥用

## 📈 性能优化

- **数据缓存**：减少API调用次数
- **异步处理**：提高并发处理能力
- **响应压缩**：减少数据传输量
- **CDN加速**：静态资源CDN分发
